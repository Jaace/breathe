# `docs/` — landing page sources

This folder is the source for the public site at
<https://jaace.github.io/breathe/>, served straight off `main` by GitHub
Pages (Settings → Pages → Source: `Deploy from a branch`, Branch:
`main`, Folder: `/docs`).

## Files

| File         | Purpose                                                           |
| ------------ | ----------------------------------------------------------------- |
| `index.html` | The single-page site. Inline CSS for tweaks; relies on the shared `https://jaace.github.io/theme/` stylesheet + theme toggle for the family look. |
| `demo.tape`  | [VHS](https://github.com/charmbracelet/vhs) script that records the hero gif. |
| `demo.gif`   | Generated by `vhs docs/demo.tape`. Not present until first recording. |

## Re-record the demo

```bash
brew install vhs        # one-time; same Charm family as bubbletea/glamour/lipgloss
vhs docs/demo.tape      # writes docs/demo.gif
```

Then commit both `demo.tape` (if you tweaked it) and `demo.gif`.

## Swap the placeholder for the real gif

Once `demo.gif` exists, edit `index.html`: in the
`<div class="terminal-section">` block, comment out the
`.demo-placeholder` div and uncomment the `<img class="demo-image">`
tag right below it.
